# Sistema de Digitalización del Centro Emmanuel

Sistema de gestión integral para la administración y digitalización de los procesos del Centro Emmanuel.

## 📋 Descripción

Este proyecto tiene como objetivo centralizar y optimizar la gestión administrativa y operativa del Centro Emmanuel mediante una plataforma web moderna.

Entre sus funcionalidades previstas se incluyen:

- Gestión de usuarios y autenticación.
- Gestión de insumos y productos elaborados.
- Control de stock.
- Registro de ventas.
- Reportes y estadísticas.
- Acceso desde dispositivos móviles.
